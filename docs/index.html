Course Listing

  • A Taste of Python for Experienced Developers
    Python Introduction
    keyboard_arrow_down
    • Learn the basic language features (and syntax) of Python: Objects, Looping, Conditionals, List, Dictionaries, Functions
    • Be able to use:
      • The Python interpreter,
      • IPython as a more feature-full interpreter,
      • Jupyter as the next-generation of IPython,
      • and stand-alone python scripts.
    • Be able to write simple Python scripts (e.g. to replace bash scripts or other simple scripts).
  • The Python Ecosystem for Full-scale Apps
    Python Introduction
    keyboard_arrow_down
    • Discover the broad Python ecosystem which facilitate building full-scale applications in Python, e.g.:
      • `modules` and `packages`,
      • virtual environment,
      • testing frameworks,
      • application frameworks (for web and otherwise),
      • packaging systems, and
      • data science toolkits and the “PyData stack”.
  • Python Looping Idioms
    Python Intermediate
    keyboard_arrow_down
    • Learn the idiomatic ways to loop in Python: enumerate, zip, sorted, reversed, itertools.
    • Build their own iterable objects (the `__iter__` and `__next__` methods).
    • Use and build generator (functions that `yield`).
    • Learn generator expressions and list comprehensions.
  • Python Ideologies
    Python Intermediate
    keyboard_arrow_down
    • Learn the most common coding conventions from both PEP8 and Google’s Python Style Guide.
    • Learn a little what it means to be “Pythonic”: The Python Manifesto, Coding with Intent, writing “Beautiful” code.
  • Python Builtin Modules and Data Structures
    Python Intermediate
    keyboard_arrow_down
    • Learn the features of the `collections` builtin module: defaultdict, Counter, namedtuple, deque, etc.
    • Be able to call low-level OS function through the builtin `os` module.
    • Be able to read/write files with various formats: XML, JSON, CSV, MS Excel, SQLite, pickle.
    • Learn how to manipulate strings and dates in Python.
  • Python Functions: First-class-ness, Callbacks, Closures, and Decorators
    Python Intermediate
    keyboard_arrow_down
    • Understand functions as a first-class type in Python:
    • Be able to pass functions as parameters to other functions (e.g. sort, pd.apply, etc.)
    • Understand and be able to write function closures:
    • Be able to return new functions from another function.
    • Understand and be able to write function decorators:
    • Understand decorators as a design pattern.
    • Understand Python’s decorator syntax (“syntactic sugar”).
  • Python Classes and Inheritance for the Experienced Developer
    Python Advanced
    keyboard_arrow_down
    • Learn Python’s class syntax and scoping rules (not what you’d expect!)
    • Understand the difference between statically-typed languages and dynamically-typed languages, and be able to articulate the pros and cons of each.
    • Use Python’s dynamic typing system to write reusable functions (embrace “duck typing”).
    • Learn (a handful of) Python’s magic methods.
    • Learn when inheritance makes sense in Python.
  • Python Case Study: Write a DataFrame object using only Python builtins!
    Python Advanced
    keyboard_arrow_down
    • Case study requirements:
      • Use only Python’s builtin modules, data structures and language features.
      • Build a `DataFrame` class (similar to Panda's, but with your own spin on things).
      • Use magic methods to create a flexible and readable interface.
      • Write documentation about what features your DataFrame supports, and what features it does not support.
  • Packaging in Python for Internal and External Distribution
    Python Advanced
    keyboard_arrow_down
    • Learn the differences between `conda-forge` and `PyPI`.
    • Learn how to use Conda to package Python for both internal and/or external distribution.
  • Python C-extensions and the GIL
    Python Advanced
    keyboard_arrow_down
    • Write, compile, and use a C-extension from scratch.
    • Wrap, compile, and use an existing C-library.
    • Learn what the GIL is, why it exists, and how to parallelize code despite the GIL (i.e. how to release the GIL in your C-extension to allow parallelism).
  • Scientific Computing in Python: Overview
    Data Science Introduction
    keyboard_arrow_down
    • Explore, learn, and practice:
      • SciPy for so many things,
      • NumPy for n-dimensional arrays,
      • Pandas for DataFrames,
      • StatsModels for … you guessed it, statistical models!
  • Scientific Computing in Python: Fast Numeric Computation
    Data Science Intermediate
    keyboard_arrow_down
    • Explore, learn, and practice:
      • vectorized operations in NumPy,
      • avoiding inefficient combinations of operations with NumPy,
      • distributed computation with Dask (introduction only),
      • GPU computation with Numba (introduction only).
  • NumPy Full Tour
    Data Science Intermediate
    keyboard_arrow_down
    • Explore the entire NumPy API (very quickly!) in order to obtain a gut-feeling for it’s overall capabilities (don’t reinvent the wheel).
  • Statistics with Python: Hypothesis Testing and Confidence Intervals
    Data Science Introduction
    keyboard_arrow_down
    • Be able to recite precisely the answer to “What actually is a p-value?”
    • Understand the tenets of random sampling and hypothesis testing, and understand its strengths and weaknesses.
    • Be able to compute confidence intervals and understand how they relate to hypothesis testing.
    • Be able to compute p-values, confidence intervals, and run hypothesis tests in Python with SciPy.
  • Statistics with Python: Random Variables and Common Distributions
    Data Science Intermediate
    keyboard_arrow_down
    • Learn the concept of random variables.
    • Understand the difference between continuous random variables and discrete random variables.
    • Understand the the common probability distributions (normal, uniform, gamma, poison, bernoulli, binomial, etc).
    • Be able to compute PDFs, PMFs, CDFs, etc using Python with SciPy.
    • Use statsmodels to do linear regression and regression analysis.
  • Statistics with Python: Modern Methods
    Data Science Advanced
    keyboard_arrow_down
    • Learn and practice:
      • Bootstrapping,
      • Bayesian statistics,
      • Why they are “modern” methods.
  • Exploratory Data Analysis with Pandas: Baisc Operations
    Data Science Introduction
    keyboard_arrow_down
    • Learn to read data into Pandas DataFrames from common formats: CSV, TSV, JSON, XML, HTML, HDF5, SAS, Excel
    • Be able to convert to the proper datatypes, and view summary statistics of a DataFrame.
    • Learn to filtering data using Series objects, indexing, and boolean masks.
    • Be able to merge/append/concatenate DataFrames.
    • Be able to export data to common formats: CSV, TSV, JSON, XML, HTML, HDF5, SAS, Excel
    • Learn how to create basic plots through Pandas.
  • Exploratory Data Analysis with Pandas: Transformation and Aggregation
    Data Science Intermediate
    keyboard_arrow_down
    • Be able to transform data within DataFrames and Series (including datetime and string transformations).
    • Understand and use the group-by operations and workflow of:
      • Aggregation
      • Transformation
      • Filtration
  • Exploratory Data Analysis with Pandas: Optimizations, Milti-index, and Time Series
    Data Science Advanced
    keyboard_arrow_down
    • Learn and practice:
      • Why vectorized operations are fast and should always be used.
      • How to recognize and avoid n^2 operations (easy to accidentally do with Pandas; kills performance).
      • The intricacies of the multi-index DataFrames.
      • How to store time-series in Pandas (and date/time manipulation and resampling).
  • Plotting in Python with Matplotlib
    Data Science Introduction
    keyboard_arrow_down
    • Learn and practice:
      • Two interfaces to Matplotlib: global, OOP
      • Seaborn: A convenience layer atop Matplotlib
      • Plotting through Pandas: Another convenience layer
  • Plotting in Python with Bokeh
    Data Science Intermediate
    keyboard_arrow_down
    • Learn the capabilities of the Bokeh plotting library.
    • Use Bokeh to create interactive plots in Jupyter.
    • Use Bokeh to create interactive plots to publish on the Web.
  • Plotting in Python with Specialized Libraries
    Data Science Advanced
    keyboard_arrow_down
    • Create plots using other plotting libraries:
      • Datashader
      • GeoViews
    • Using PyViz as a high-level plotting library.
